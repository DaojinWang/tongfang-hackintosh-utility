# Project: STAR BEAT!

Next generation Tongfang Hackintosh Utility. (WIP)

## Features

- Keyboard light controller
- Manage and customize OpenCore config
- Useful scripts (Fix sleep, Enable HiDPI, etc.)
- More friendly User Interface
- Internationalization
